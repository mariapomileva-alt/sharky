For your team share

  • Give them one link — after you publish this site (e.g. GitHub Pages), send the main page URL in WhatsApp/Telegram. No app install.
  • Weekend cups / results — they open admin-results.html with the PIN you set in that file. Data lives in each browser unless you export / import JSON there so everyone matches.
  • Trip bus seats — same link; with useCloud: true in bus-config.js + Supabase, everyone sees one bus. Otherwise seats are only on that device.
  • Where awards are stored — in the browser’s localStorage key for results, or in the JSON file you share from admin; optional cloud for the bus is in your Supabase table bus_board.
